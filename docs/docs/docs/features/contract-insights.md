# Smart Contract Insights

CasperLens provides visualizations for insights on frequency of usage of a contract version for transactions.

This provides observability capabilities that Casper Network Explorer does not have

::: warning Warning
The insights are reliant on transactions retrieval, which may experience rate limiting on higher number of transactions. Thus, analysis is limited on basis of hours.
:::
