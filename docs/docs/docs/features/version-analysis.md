# Version Analysis

CasperLens allows on-demand version analysis between any 2 versions of a contract and provides the modifications or changes observed between them.

It provides details on additions, modification and deletion to entry keys and named keys if any.

The difference can also be analyzed using AI for a summarized explanation of implications of the difference.
