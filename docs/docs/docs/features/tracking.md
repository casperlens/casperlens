# Smart Contract Tracking

Multiple smart contracts can be tracked under the same account for more information on lifecycle, diff and transaction history.

They can be accessed at the dashboard.

::: info Note
The tracked contracts will remain accessible as long as the user does not clear session data or cookies. The session account if logged out of, will make the tracked contracts inaccessible.
:::
